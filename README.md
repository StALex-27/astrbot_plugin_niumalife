# 牛马人生 - AstrBot 打工群文字模拟经营游戏

![version](https://img.shields.io/badge/version-0.0.1-blue)
![AstrBot](https://img.shields.io/badge/AstrBot-Plugin-green)

**牛马人生** 是一款嵌入 AstrBot 的群聊文字模拟经营游戏。玩家通过打工、学习、娱乐、饮食等行为体验"牛马"生活，积累金币、提升技能、购置房产、投资股市。

---

## 🎮 功能概览

### 核心系统

| 系统 | 说明 | 指令 |
|------|------|------|
| **签到** | 每日签到获金币，有概率触发幸运掉落（buff/食物/金币） | `/签到` |
| **档案** | 查看角色属性、技能、buff、签到记录 | `/档案` |
| **打工** | 选择工作（体力/脑力），消耗体力精力，赚取金币 | `/打工 [工作名] [小时]` |
| **学习** | 消耗金币学习课程，获得经验提升技能等级 | `/学习 [课程名] [小时]` |
| **娱乐** | 消耗金币恢复心情 | `/娱乐 [名称] [小时]` |
| **饮食** | 购买食物恢复各项属性 | `/吃 [食物名]` |
| **住所** | 租房或购房，不同住所影响被动恢复效率 | `/住 [租/买] [名称]` |
| **股市** | 模拟股市，低买高卖赚取差价 | `/股市` |
| **背包** | 查看和管理获得的物品/buff | `/背包` |
| **取消** | 取消当前进行中的动作 | `/取消` |

### 属性系统

| 属性 | 说明 | 异常影响 |
|------|------|---------|
| ❤️ 健康 | 角色基础健康值 | <50 提示警告 |
| 💪 体力 | 影响体力工作产出 | 不足无法工作 |
| ⚡ 精力 | 影响脑力工作产出 | 不足无法学习 |
| 😊 心情 | 影响所有活动效率 | <20 警告 |
| 🍖 饱食 | 每分钟自然消耗 | <20 健康/心情下降 |

### 工作类型

**体力型**：搬砖、清洁工、外卖配送、快递分拣、工地小工  
**脑力型**：文员、数据录入、客服、会计、程序员  
**技能型**：销售、项目经理、技术总监、企业顾问

### 住所系统

| 类型 | 示例 | 效果 |
|------|------|------|
| 免费 | 桥下 | 基础恢复 |
| 租用 | 地下室、合租房、公寓 | 日租，更高恢复加成 |
| 购买 | 景江公寓、别墅、海豹国际 | 一次性购买，最高恢复加成 |

## ⚙️ 工作机制

### Tick 系统（后台自动运行）

| 循环 | 周期 | 功能 |
|------|------|------|
| 主循环 | 每分钟 | 被动恢复、饱食消耗、夜间自动睡眠检测 |
| 小时循环 | 每小时 | 数据保存 |
| 每日结算 | 23:30 | 房租扣除、异常状态结算 |

### 自动睡眠
- 0点-8点期间，处于**空闲**状态的用户自动进入睡眠
- 睡眠时长根据住所加成计算

### 每日结算
- 扣除当日房租
- 饱食度-20，过低则健康/心情下降

## 🎨 渲染输出

插件使用 **HTML 卡片渲染**，通过 AstrBot 内置 `html_renderer.render_custom_template()` 生成图片卡片，包含：

- 档案卡片 / 状态卡片
- 签到卡片（含幸运评级展示）
- 工作/学习/娱乐开始卡片
- 食物/住所/娱乐列表卡片
- 股市卡片
- 错误/成功提示卡片
