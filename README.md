# 牛马人生 - AstrBot 打工群文字模拟经营游戏

![version](https://img.shields.io/badge/version-0.1.7-blue)
![AstrBot](https://img.shields.io/badge/AstrBot-Plugin-green)

**牛马人生** 是一款嵌入 AstrBot 的群聊文字模拟经营游戏。玩家通过打工、学习、娱乐、饮食等行为体验"牛马"生活，积累金币、提升技能、购置房产，投资股市。

---

## 🎮 功能概览

| 系统 | 说明 | 指令 |
|------|------|------|
| **签到** | 每日签到获金币，有概率触发幸运掉落（buff/食物/金币） | `/签到` |
| **档案** | 查看角色属性、技能、buff、签到记录 | `/档案` |
| **打工** | 选择工作（体力/脑力），消耗体力精力，赚取金币 | `/打工 [工作名] [小时]` |
| **学习** | 消耗金币学习课程，获得经验提升技能等级 | `/学习` |
| **娱乐** | 消耗金币恢复心情 | `/娱乐 [名称] [小时]` |
| **饮食** | 购买食物恢复各项属性 | `/吃 [食物名]` |
| **住所** | 租房或购房，不同住所影响被动恢复效率 | `/住 [租/买] [名称]` |
| **股市** | 模拟股市，低买高卖赚取差价 | `/股市` |
| **背包** | 查看和管理获得的物品/buff | `/背包` |
| **取消** | 取消当前进行中的动作 | `/取消` |

## 📚 学习系统 v3

### 机构一览

| 机构 | 层级 | 定位 |
|------|------|------|
| 🏫 匠心学堂 | T1基础 | 语数英理化生史地音美，基础课程 |
| 🔧 技能职校 | T1基础 | 苦力/驾驶/烹饪/理财/办公/沟通/体育 |
| 💻 IT学院 | T1+T2+IT_T3 | 计算机基础→编程→网络→T3导论 |
| 💼 商学院 | T1+T2+商_T3 | 会计/营销/销售/电商/人力/物流 |
| 🎨 艺体学院 | T1+T2+艺_T3 | 音乐/美术/表演/动画/体育/写作 |
| 📚 文理学院 | T1+T2+文_T3 | 英语/翻译/法律/心理/播音主持 |
| ⭐ 专精堂 | T2+T3专精 | 大师之路，高阶专项突破 |

### 技能三阶体系

| 阶 | 技能数 | 经验曲线 | 满级经验 | 特点 |
|----|--------|---------|---------|------|
| T1 基础 | 18个 | fast（速成） | 3600 | 无前置，约30天满级 |
| T2 专业 | 20个 | standard（标准） | 5400 | 需前置，约45-60天满级 |
| T3 专精 | 18个 | mastery（精修） | 8100 | 需T2前提，约60-90天满级 |

### 学习指令

```
/学习              → 推荐3-4所相关机构 + 可学课程
/学习 查询         → 所有机构列表
/学习 查询#IT学院 → IT学院详情 + 全部课程
/学习 查询#编程    → 编程技能全路径（所有机构+课程）
/学习 课程名       → 直接开始学习（固定2小时，直接扣费）
```

## ⚙️ 工作机制

### Tick 系统（后台自动运行）

| 循环 | 周期 | 功能 |
|------|------|------|
| 主循环 | 每分钟 | 被动恢复、饱食消耗、夜间自动睡眠检测 |
| 小时循环 | 每小时 | 数据保存 |
| 每日结算 | 23:30 | 房租扣除、异常状态结算 |

### 自动睡眠
- 0点-8点期间，处于**空闲**状态的用户自动进入睡眠
- 睡眠时长根据住所加成计算

### 每日结算
- 扣除当日房租
- 饱食度-20，过低则健康/心情下降

## 🎨 渲染输出

插件使用 **HTML 卡片渲染**，通过 AstrBot 内置 `html_renderer.render_custom_template()` 生成图片卡片，包含：

- 档案卡片 / 状态卡片
- 签到卡片（含幸运评级展示）
- 工作/学习/娱乐开始卡片
- 食物/住所/娱乐列表卡片
- 股市卡片
- 错误/成功提示卡片
